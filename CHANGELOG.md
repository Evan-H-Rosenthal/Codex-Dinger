# Changelog

All notable changes to Codex Dinger are documented here.

## 0.2.0 - 2026-07-17

- Add natural-language update checks and installation through a bundled skill.
- Refresh only the Codex Dinger GitHub marketplace during update checks.
- Report installed and available versions before changing the installed bundle.

## 0.1.0 - 2026-07-17

- Play a hidden notification sound from the main-task `Stop` hook.
- Include a distributable default chime.
- Add local sound-bank, selection, preview, volume, and enable/disable controls.
- Add a Codex skill for natural-language sound management.
